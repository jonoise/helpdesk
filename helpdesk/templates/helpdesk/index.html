Mi versión a pequeña escala de un sistema Helpdesk para subir issues por medio de tickets. Este es mi segundo proyecto, y fue muchísimo más "relax" que mi primer proyecto deployed. Aquí desarrollé un sistema simple que trata de asemejar algunas de las principales características de un sistema a escala empresarial. Apliqué muchísimo de lo que aprendí desarrollando mi portfolio website, por ejemplo: estructurar el proyecto desde el principio hasta el final antes de escribir una línea de código para aminorar contratiempos y eventualidades. También implementé otras cosas que fui recogiendo en el proceso, como por ejemplo hacer la primer migración con un AbstractUser (leer account.models.MyUser Docstring).

Pueden leer el README. O pueden leer y clonar el proyecto desde Github.
